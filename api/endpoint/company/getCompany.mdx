# Get company details with specific fields

```bash cURL
curl -X GET "https://eqs-platform-api.onrender.com/v1/company?fields=name,email,currency,subscription" \
  -H "Authorization: API-Key YOUR_API_KEY"
```

```javascript Node.js
import fetch from "node-fetch";

const params = new URLSearchParams({
  fields: "name,email,currency,subscription",
});

fetch(`https://eqs-platform-api.onrender.com/v1/company?${params.toString()}`, {
  headers: {
    Authorization: `API-Key ${process.env.EQUIS_API_KEY}`,
  },
})
  .then((res) => res.json())
  .then((json) => console.log(json))
  .catch((err) => console.error("Error:", err));
```

## Query Parameters

{" "}

<ParamField query="fields" type="string">
  A comma-separated list of company fields to return. For example:
  `fields=name,email,currency,subscription`.
</ParamField>
<Note>
  {" "}
  Any query parameters other than `fields` will result in a `400 Bad Request` error.{" "}
</Note>
