<RequestExample>

```bash cURL
# Get the first page of remittances (default 10)
curl -X GET "https://eqs-platform-api.onrender.com/v1/remittances" \
  -H "Authorization: API-Key YOUR_API_KEY"
```

# Get remittances with specific fields and custom pagination

```bash cURL
curl -X GET "https://eqs-platform-api.onrender.com/v1/remittances?page=2&limit=20&fields=remittanceId,amount,status&viewType=legacy" \
  -H "Authorization: API-Key YOUR_API_KEY"
```

```javascript Node.js
import fetch from "node-fetch";

const params = new URLSearchParams({
  page: 1,
  limit: 15,
  fields: "remittanceId,amount,status,paymentDate",
  viewType: "all",
});

fetch(`https://eqs-platform-api.onrender.com/v1/remittances?${params.toString()}`, {
  headers: {
    Authorization: `API-Key ${process.env.EQUIS_API_KEY}`,
  },
})
  .then((res) => res.json())
  .then((json) => console.log(json))
  .catch((err) => console.error("Error:", err));
```
